## 🪄 What is this?

This repository is my personal JavaScript learning journal, following the [**Chai aur JavaScript**](https://youtube.com/playlist?list=PLu71SKxNbfoBuX3f4EOACle2y-tRC5Q37) playlist by **Hitesh Choudhary (Chai aur Code)** — a 51-video Hindi series that revisits JavaScript from the ground up with clarity and depth.

Even though I already know JavaScript, I believe true mastery begins with revisiting the fundamentals. This repo captures every concept, experiment, and "aha!" moment from that journey.

---


## 🎯 Why this repo?

- 📌 To **strengthen the foundations** I might have skipped over
- 📝 To maintain a **clean, searchable code journal**
- 🔁 To **revisit and iterate** — good code is written twice
- 🌱 To track how far I've come, one commit at a time

---

## 🧠 What I'm learning / reinforcing

- ✅ Data types and type coercion
- ✅ Memory model — Stack vs Heap
- ✅ Scope, closures, and execution context
- ✅ `this` keyword — the many faces
- ✅ Prototypes and the prototype chain
- ✅ Async JavaScript — callbacks → promises → async/await
- ✅ DOM manipulation
- ✅ Event loop and call stack internals
- ✅ Higher-order functions and functional patterns
